# Model Card: Glass-Box Diffusion Pseudotime Pipeline ## Model details**Model type:** Graph-based trajectory inference pipeline (diffusion geometry + pseudotime + terminal-based lineage proxy).  **Intended use:** Benchmarking, teaching, and rapid prototyping of branching trajectory inference for droplet scRNA-seq.  **Implementation:** Python, CPU-only, using NumPy, SciPy, scikit-learn, pandas, Matplotlib.This is not a predictive ÒmodelÓ in the supervised learning sense. It is an algorithmic analysis pipeline that produces:- inferred pseudotime for each cell,- a two-lineage assignment,- ranked gene lists for progression and branch separation,- composite figures.## Inputs- A sparse cell-by-gene UMI count matrix.- Per-cell metadata (optional), for example, known labels or batch IDs.In this repositoryÕs default run, inputs are generated by the built-in simulator.## Outputs- **Per-cell outputs**  - `dpt_pseudotime` (0Ð1 scaled)  - `inferred_lineage` (Lineage_1 or Lineage_2)- **Per-gene outputs**  - `corr_with_dpt` (geneÐpseudotime association)  - `late_mean_diff` (Lineage_1 ? Lineage_2 in late cells)- **Figures**  - diffusion space plots, QC plots, gene program panels  - composite multi-panel figures## Training dataNot applicable. No training occurs.  The pipeline is deterministic given:- simulation parameters and random seed, or- a fixed input dataset.## EvaluationFor simulated data with ground truth:- Agreement between true and inferred pseudotime is reported as Pearson correlation.- Topology is qualitatively assessed using diffusion space structure and lineage separation.- Gene recovery is assessed by correlation spectra and branch effect sizes.For real data:- There is no intrinsic ground truth. Recommended evaluation includes:  - robustness to kNN/kernels,  - concordance with known marker genes,  - alignment with independent measurements (e.g., time series, perturbations).## Intended use- Trajectory inference demonstrations.- Benchmark harness for comparing:  - normalization strategies,  - graph construction choices,  - diffusion component dimensionality,  - lineage assignment methods.- Generation of reproducible outputs from a standardized run.## Out-of-scope use- Clinical decision-making.- Claims about causality of gene programs without experimental validation.- Using inferred pseudotime as a surrogate for real chronological time without supporting evidence.## Ethical considerations- If applied to human data, ensure appropriate consent, governance, and privacy safeguards.- Trajectory inference can be over-interpreted. Branch structure may reflect confounders (batch, cell cycle, stress) rather than true biology.## Limitations and failure modes- Scalar pseudotime compresses branching, branchpoint cells can be ambiguous.- Terminal-based two-lineage proxy is a simplification and may fail for multi-branch or looping topologies.- Graph construction is sensitive to preprocessing choices and parameter settings.- PCA-based visualization is a lightweight approximation and may under-represent non-linear structure.## Key references - Svensson, V.
